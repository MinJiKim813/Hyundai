# 🚗 Hyundai Clone

현대자동차 공식 홈페이지를 클론 코딩한 프로젝트입니다.

HTML, CSS, JavaScript를 활용하여 PC와 Tablet, Mobile 환경에 대응하는 반응형 웹을 구현했습니다. 디바이스별 레이아웃 변화와 다양한 사용자 인터랙션을 구현하며 사용자 경험을 고려한 웹 퍼블리싱을 연습했습니다.

## 💭 Project Retrospective

반응형 웹을 구현하며 화면 크기에 따라 레이아웃과 사용자 인터랙션이 달라지는 과정을 직접 경험할 수 있었습니다.

특히 동일한 콘텐츠를 다양한 디바이스에서 자연스럽게 사용할 수 있도록 레이아웃을 구성하고, 터치와 드래그를 포함한 사용자 인터랙션을 구현하면서 반응형 웹 퍼블리싱에 대한 이해를 높일 수 있었습니다. 또한 디바이스별 사용성을 고려하며 UI를 구현하는 과정에서 사용자 경험의 중요성을 다시 한번 느낄 수 있었습니다.
